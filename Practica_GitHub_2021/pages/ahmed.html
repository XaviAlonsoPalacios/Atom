MANUAL COMANDOS GIT - Curs 2020/2021

Explicació_Comando. git branch "nombre": el comando git branch "nombre" creará una rama nueva con el nombre deseado en el repositorio aparte de la rama master. Ejemplo: git branch atom; ejecutando este comando creará rama nombrado atom.Es decir atom es el nombre de la rama creado
